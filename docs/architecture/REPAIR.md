# Repair and Rematching

Repair operates only on unresolved, broken, or explicitly selected entries.

## Rules

- Preserve original CBL metadata and previous mapping history.
- Do not rebuild unaffected entries.
- Allow source replacement for a complete series while retaining entry-level exceptions.
- Preview changes before replacing automatically matched entries.
- Never replace user-confirmed mappings without explicit approval.
- Distinguish an unavailable extension from a missing remote chapter.
- Reuse cached confirmed series mappings when still valid.
