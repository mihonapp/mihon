# CBL Import Flow

1. Select a local `.cbl` file or explicitly enter a remote CBL URL.
2. Parse safely and show list name, entry count, and validation warnings.
3. Select installed compatible extensions and rank them for this list.
4. Configure or accept the confidence and ambiguity defaults.
5. Resolve distinct series groups.
6. Resolve issue entries from fetched chapter lists.
7. Review ambiguous or low-confidence candidates.
8. Save the reading list and mappings.
9. Optionally add matched series to the normal library or a category.
10. Open the reading-list overview, where every entry remains editable.

The import must be resumable after cancellation or source failure without discarding completed user confirmations.
